# mlunkeit.dev

Welcome to my personal portfolio website and digital notebook. This project is a minimalist web application built to showcase my projects
and host my notes on mathematics, cryptography and computer science related topics.

The design is heavily inspired by the **Catppuccin** color palette and clean, monospace-driven CLI interfaces.

## Tech Stack

- **Framework**: [React](https://react.dev) + [Vite](https://vitejs.dev)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Theme**: [Catppuccin](https://catppuccin.com) Mocha (dark) and Latte (light)
- **Icons**: Lucide React + devicons.io
